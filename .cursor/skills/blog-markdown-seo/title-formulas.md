# SEO title formulas

Use **different formulas** for each candidate so the set spans intents and angles. Adapt templates to pt-BR; keep natural syntax, not keyword stuffing.

## Core formulas

| ID | Formula | When to use | Template (pt-BR) |
|----|---------|-------------|------------------|
| **outcome** | Result + mechanism | Strong proof or surprising result | `[Resultado]: como [mecanismo] [verbo]` |
| **how-to** | Explicit guide | Tutorial or reproducible process | `Como [ação] com [ferramenta/conceito]` |
| **question** | Search / PAA alignment | Reader has an open question | `[Pergunta]? [contexto curto]` |
| **contrast** | Two worlds bridged | Essay linking domains | `[A] e [B]: [tese em poucas palavras]` |
| **number** | Quantified credibility | Lists, streaks, metrics | `[N] [coisas/vezes/lições] sobre [tópico]` |
| **year** | Freshness signal | News, events, annual topics | `[Tópico] em [ano]: [ângulo]` |
| **audience** | Role-specific | Devs, founders, practitioners | `[Papel]: [benefício ou alerta] sobre [tópico]` |
| **mistake** | Negative framing | Pitfalls (use honestly) | `[Erro comum] em [tópico] (e o que fazer)` |

## Keyword placement

1. Identify **primary keyword phrase** (2–5 words): e.g. `simulação EA Sports`, `modelagem Monte Carlo`.
2. Place it in the **first 40 characters** when grammar allows.
3. Add **one secondary term** in title or description (not both stuffed): e.g. `probabilidade`, `oráculo computacional`.
4. Prefer phrasing people actually search; avoid internal jargon unless the post defines it.

## Intent mapping

| Intent | Title signals | Description should |
|--------|---------------|-------------------|
| Informational | "O que é", "Por que", question titles | Summarize insight + scope |
| How-to | "Como", "Guia", "Passo a passo" | State outcome + prerequisite |
| Opinion / essay | Contrast, thesis, named concept | Clarify stance without spoiling |
| News / reaction | Year, event name, "previu", "anunciou" | Date/context + why it matters |

## Power words (sparingly)

Use at most **one** per title: `previu`, `revela`, `explica`, `guia`, `por trás de`, `na prática`, `além de`. Avoid stacks like "incrível segredo definitivo".

## Anti-patterns

- **Vague tease:** "Você não vai acreditar…" without substance in the post.
- **Duplicate title/description:** same sentence twice in SERP.
- **Keyword list:** "IA, ML, dados, futuro" with no promise.
- **All caps or emoji** in `title` frontmatter (hurts readability in RSS and some SERP renderers).
- **Title wider than article:** promising a comparison the post does not make.

## Scoring (pick winners)

Rate each candidate 1–5 on:

1. **Intent match** — Does it fit what the post actually does?
2. **Keyword fit** — Primary phrase present and natural?
3. **Specificity** — Concrete nouns, numbers, or constraints?
4. **SERP clarity** — Understandable if truncated at ~60 characters?
5. **Voice** — Fits encodere.se / blog tone (smart, not spammy)?

Recommend the top 1–2; note tradeoffs (e.g. more clicks vs more accurate).
